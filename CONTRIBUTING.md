# Contributing to Byte Coder Ai Agent

Thank you for your interest in contributing! We welcome improvements to make this agent even better.

## How to Contribute

1.  **Fork the Repository**: click the "Fork" button on GitHub.
2.  **Clone your Fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/byte-coder-ai-agent.git
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/my-new-feature
    ```
4.  **Make Changes**: Improve the code, fix bugs, or add features.
5.  **Test**: Ensure `npm run compile` passes and the extension runs correctly in `F5` debug mode.
6.  **Commit**:
    ```bash
    git commit -m "feat: Add amazing new feature"
    ```
7.  **Push**:
    ```bash
    git push origin feature/my-new-feature
    ```
8.  **Open a Pull Request**: Submit your PR on the main repository.

## Coding Standards

- **TypeScript**: We use strict TypeScript. Ensure no `any` types unless absolutely necessary.
- **Formatting**: Keep code clean and readable.
- **Comments**: Document complex logic.

## Reporting Issues
If you find a bug, please open an issue on GitHub with steps to reproduce.
